26. jelaskan maksud kode berikut int array 1 [5] = {7,6,9,1,16} ;​


Jawaban:

Kode tersebut merupakan inisialisasi dan deklarasi dari sebuah array dengan nama "array1" yang memiliki ukuran 5 dan berisi nilai-nilai {7, 6, 9, 1, 16}.

Arti dari kode tersebut adalah sebagai berikut:

- "int" merupakan tipe data dari elemen-elemen dalam array, dalam hal ini bertipe integer (bilangan bulat).

- "array1" adalah nama dari array tersebut. Nama ini bisa digunakan untuk mengakses elemen-elemen dalam array.

- "[5]" menunjukkan bahwa array memiliki ukuran 5, artinya terdiri dari 5 elemen.

- "{7, 6, 9, 1, 16}" adalah nilai-nilai yang disimpan di dalam array. Dalam hal ini, array memiliki 5 elemen dengan nilai berturut-turut 7, 6, 9, 1, dan 16.

Jadi, dengan kode tersebut, kita telah membuat dan menginisialisasi array "array1" dengan 5 elemen yang berisi nilai-nilai tertentu. Misalnya, kita bisa mengakses nilai 6 dengan menggunakan "array1[1]", dan nilai 16 dengan "array1[4]".

29. Berikan contoh fungsi dalam bahasa C untuk proses yang ada dalam Linear Queue berbasis array dan berikan penjelasannya


Implementasi Fungsi (Bahasa C) untuk Proses dalam Linear Queue berbasis Array

Sebagai asumsi awal, pada program telah didefinisikan hal-hal berikut ini.

Konstanta [tex]\texttt{MAX\_QUEUE\_SIZE}[/tex], yang menyatakan kapasitas maksimum Queue.[tex]\texttt{int queue[MAX\_QUEUE\_SIZE]}[/tex] sebagai definisi Queue.Variabel [tex]\tt front[/tex] dan [tex]\tt rear[/tex] bertipe integer, yang masing-masing diinisialisasi dengan nilai 0 dan –1.

Proses utama pada Queue adalah enqueue dan dequeue. Sebelum membahas implementasi kedua proses utama tersebut, kita dapat melakukan implementasi beberapa fungsi/prosedur pendukung, yaitu:

apakahQueueKosong(), atau isQueueEmpty(), yang mengembalikan nilai 0 (false) jika Queue tidak kosong, dan nilai 1 (true) jika Queue kosong.apakahQueuePenuh(), atau isQueueFull(), yang mengembalikan nilai 0 (false) jika Queue tidak penuh, dan nilai 1 (true) jika Queue penuh.printQueue(), untuk mencetak isi Queue pada perangkat output (stdout atau layar).Fungsi-Fungsi Pendukungint apakahQueueKosong() {
    return rear == -1;
}
Penjelasan: Queue dinyatakan kosong jika variabel [tex]\tt rear[/tex] bernilai –1.int apakahQueuePenuh() {
    return rear == MAX_QUEUE_SIZE-1;
}
Penjelassan: Queue dinyatakan penuh jika variabel [tex]\tt rear[/tex] bernilai 1 kurangnya dari nilai konstanta kapasitas maksimum Queue.void printQueue() {
    printf("Queue = ");
    if (apakahQueueKosong())
       printf("KOSONG.");
    else {
        for (int i = front; i <= rear; i++)
            printf("%d ", queue[i]);
    }
    printf("\n");
}
Penjelasan: Jika Queue kosong, cetak “KOSONG“. Namun jika tidak, lakukan iterasi pencetakan elemen Queue dari indeks [tex]\tt front[/tex] hingga indeks [tex]\tt rear[/tex].Fungsi Proses Utama: Enqueue dan Dequeue

void enqueue(int data) {
    if (apakahQueuePenuh()) {
        printf("ENQUEUE %d gagal. Queue dalam keadaan penuh.\n", data);
        return;
    }
    rear++;
    queue[rear] = data;
    printf("%d masuk ke dalam Queue.\n", data);
}

Untuk menambahkan elemen pada Queue, dilakukan analisis kasus apakah Queue dalam keadaan penuh, dengan fungsi [tex]\tt apakahQueuePenuh()[/tex].

Jika Queue penuh, cetak pesan bahwa enqueue gagal, dan keluar dari fungsi.Sebaliknya, jika Queue belum penuh, lakukan inkrementasi variabel [tex]\tt rear[/tex], dan elemen/data dimasukkan ke dalam Queue, pada indeks [tex]\tt rear[/tex] (indeks paling belakang dari Queue).

Kita tidak perlu melakukan apa-apa terhadap variabel [tex]\tt front[/tex], karena elemen pertama Queue tetap berindeks 0.

int dequeue() {
    if (apakahQueueKosong()) {
        printf("DEQUEUE gagal. Queue dalam keadaan kosong.\n");
        return INT_MIN;
    }
    int data = queue[front];
    for (int i = front; i <= rear; i++) {
        queue[i] = queue[i+1];
    }
    rear--;
    printf("%d keluar dari Queue.\n", data);
    return data;
}

Berbeda dengan enqueue, fungsi dequeue mengembalikan nilai data yang dikeluarkan dari Queue. Sebagai langkah awal, dilakukan analisis kasus apakah Queue kosong.

Jika kosong, maka proses dequeue gagal. Tampilkan pesan dan keluar dari badan fungsi sembari mengembalikan nilai [tex]\tt INT\_MIN[/tex]. Nilai ini adalah konstanta (dari library limits.h) yang berisi nilai minimum tipe data integer pada C.Jika Queue tidak kosong, maka variabel [tex]\tt data[/tex] diisi dengan nilai elemen paling depan dari Queue, yaitu [tex]\tt queue[front][/tex]. Kemudian, dengan iterasi/looping, geser isi Queue maju 1 posisi. Setelah itu, dekrementasi variabel [tex]\tt rear[/tex].Contoh Hasil Eksekusi

Berikut ini adalah contoh implementasi program untuk menjumlahkan setiap isi sel array A dengan B dan menyimpannya dalam array C:

# deklarasi array A, B, dan C

A = [[0 for i in range(4)] for j in range(5)]

B = [[0 for i in range(4)] for j in range(5)]

C = [[0 for i in range(4)] for j in range(5)]

# inisialisasi isi array A dan B

for i in range(4):

for j in range(5):

A[i][j] = i + j

B[i][j] = i * j

# jumlahkan setiap elemen dari array A dan B

# dan simpan hasilnya dalam array C

for i in range(4):

for j in range(5):

C[i][j] = A[i][j] + B[i][j]

# cetak isi array C

for i in range(4):

for j in range(5):

print(C[i][j], end=" ")

print()
