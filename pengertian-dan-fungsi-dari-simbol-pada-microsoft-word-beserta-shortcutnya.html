18. Jelaskan masing-masing fungsi dari simbol berikut pada Microsoft Word ( +, -, *, /, ^ ) !


Jawaban:

Saya jawab sesuai foto soal kak (pada Ms. Excel):

+ ialah simbol operasi penjumlahan angka

- ialah simbol operasi pengurangan angka

* ialah simbol operasi perkalian angka

/ ialah simbol operasi pembagian angka

^ ialah simbol operasi perpangkatan angka

Penjelasan tambahan:

Perlu dimengerti bahwa Ms. Excel ialah salah 1 program pengolah data angka, sehingga simbol² di atas memudahkan pengguna program untuk mengolah data² angka yg sudah diinput dalam program itu.
