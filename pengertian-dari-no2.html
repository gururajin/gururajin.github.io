2. kak tolong pengertiannyaApa itu gas NO2, CO, dan SO2?​


Jawaban:

NO2 : Nitrogen dioksida adalah senyawa kimia dengan rumus NO ₂. Satu dari beberapa oksida nitrogen, NO ₂ digunakan sebagai bahan sintesis untuk pembuatan asam nitrit, yang produksinya mencapai jutaan ton tiap tahunnya.

CO : Karbon dioksida atau zat asam arang adalah sejenis senyawa kimia yang terdiri dari dua atom oksigen yang terikat secara kovalen dengan sebuah atom karbon. Ia berbentuk gas pada keadaan temperatur dan tekanan standar dan hadir di atmosfer bumi.

SO2 : Belerang dioksida adalah senyawa kimia dengan rumus SO ₂. Senyawa ini merupakan gas beracun dengan bau menyengat yang dilepaskan oleh gunung berapi dan beberapa pemrosesan industri.

Jumlah mol NO₂ = 163,043478 mol

Pembahasan

[tex]\boxed{\boxed{\bold{\pink{Stoikiometri}}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

[tex]\boxed{\boxed{\bold{\orange{Diket}}}}[/tex]  

massa NO₂ = 7500 gram

Ar N = 14

Ar O = 16

[tex]\boxed{\boxed{\bold{\purple{Ditanya}}}}[/tex]  

a. Jumlah mol NO₂

b. Jumlah molekul NO₂

[tex]\boxed{\boxed{\bold{\blue{Jawab}}}}[/tex]  

Menentukan Mr NO₂

Mr NO₂ = Ar N + 2. Ar O

Mr NO₂ = 14 + 2. 16

Mr NO₂ = 14 + 32

Mr NO₂ = 46

a. Jumlah mol NO₂

n NO₂ = [tex]\frac{massa}{Mr}[/tex]

n NO₂ = [tex]\frac{7500}{46}[/tex]

n NO₂ = 163,043478 mol

b. Jumlah molekul NO₂

Jumlah molekul NO₂ = n x L

Jumlah molekul NO₂ = 163,043478 x 6,02. 10²³

Jumlah molekul NO₂ = 981,521738. 10²³ molekul

[tex]\boxed{\boxed{\bold{\pink{Kesimpulan}}}}[/tex]

11. Hitung bilangan oksidasi N dalam NO3,NO2-,NH4+,NH3 dan NO2,,, !


Bilangan oksidasi N dalam NO₃⁻ ( karena tidak ada senyawa NO₃) adalah +5 . Bilangan oksidasi N dalam NO₂ ( karena tidak ada senyawa NO₂⁻ ) adalah +4 . Bilangan oksidasi N dalam NH₄⁺  adalah -3 .Bilangan oksidasi N dalam NH₃ adalah -3.Bilangan oksidasi N dalam NO₂ adalah +4.

Pembahasan

∴ Bilangan oksidasi N pada NO₃⁻ adalah +5.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya  pada NO₃ untuk memperoleh biloks N

∵Biloks total NO₃⁻ adalah -1 ( karena bilang oksidasi suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 )

NO₂ = -1

(jumlah atom N x biloks atom N) + (jumlah atom O x biloks atom O) = -1

(1 x biloks N) + (3x (-2)) = -1

Biloks N - 6 = -1

Biloks N= 6-1

Biloks N = +5

∴ Bilangan oksidasi N pada NO₂ adalah +4.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada NO₂ untuk memperoleh bilangan oksidasi N

∵Bilangan oksidasi total NO₂  adalah 0 ( karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya adalah -2 )

NO₂ = 0

(jumlah atom N x biloks atom N) + (jumlah atom O x biloks atom O) = 0

(1 x biloks N) + (2x (-2)) = 0

Biloks N - 4 = 0

Biloks N= 0+4

Biloks N = +4

∴ Bilangan oksidasi N pada NH₄⁺   adalah -3.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada NH₄⁺ untuk memperoleh biloks N

∵Biloks total   NH₄⁺  adalah +1 (  suatu ion poliatomik adalah sesuai dengan jumlah muatannya ),  dan biloks H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

Biloks  NH₄⁺   = +1

(jumlah atom N x biloks atom N) + (jumlah atom H x biloks atom H) = +1

(1 x biloks N) + ( 4 x 1) = +1

Biloks N + 4 = +1

Biloks N = +1 -4

Biloks N = -3

∴ Bilangan oksidasi N pada NH₃   adalah -3.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada NH₃ untuk memperoleh biloks N

∵Biloks total   NH₃  adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ),  dan biloks H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

Biloks  NH₃   = 0

(jumlah atom N x biloks atom N) + (jumlah atom H x biloks atom H) = 0

(1 x biloks N) + ( 3 x 1) = 0

Biloks N + 3 = 0

Biloks N = -3

Pembahasan:

Bilangan Oksidasi atau merupakan bilangan yang ditetapkan untuk suatu atom unsur dalam reaksi kimia yang menunjukkan jumlah elektron yang dilepas atau diikat (jika tanda biloks-nya negatif).

Adapun aturan-aturan penentuan bilangan oksidasi, antara lain sebagai berikut.

Bilangan oksidasi untuk atom yang berikatan bebas (tidak berikatan dengan atom unsur lain) adalah nol. Misal: .

Untuk senyawa netral, jumlah bilangan oksidasi semua atom di dalam senyawa tersebut sama dengan nol. Misal: .

Untuk ion, jumlah bilangan oksidasi atom atau semua atom sesuai dengan muatan ionnya. Misal: mempunyai biloks -2 serta untuk biloks-nya sama dengan -3.

Dalam golongan IA, unsur-unsurnya mempunyai biloks bernilai +1. Sedangkan dalam golongan IIA, unsur-unsurnya memiliki biloks bernilai +2.

Bilangan oksidasi unsur Oksigen bernilai -2, dengan pengecualian terhadap senyawa peroksida dimana biloks O sama dengan -1 dan senyawa superoksida dimana biloks O sama dengan . Misal: dan .

Bilangan oksidasi unsur Hidrogen bernilai +1, namun untuk senyawa logam hidrida biner, biloks H sama dengan -1. Misal: NaBH₄, AlH₃, TiH₂.

Bilangan oksidasi unsur Fluorin selalu bernilai -1.

28. Dalam 4,6 gram NO2 (Ar N = 14; O = 16) terdapat ... molekul NO2.


Dalam 4,6 gram NO₂ (Ar N = 14; O = 16) terdapat 6,02 × 10²² molekul NO₂.

Pembahasan

Diketahui :

gr = 4,6 gram NO₂

Ar N = 14

Ar O = 16

Ditanya :

Jumlah partikel ?

Jawab :

Pertama, kita cari nilai n (mol) terlebih dahulu.

n = gr / Mr

n = 4,6 / [14 + 2(16)]

n = 4,6 / 46

n = 0,1 mol

Langkah terakhir, mencari jumlah partikel dg men-substitusikan nilai mol ke rumus. Ingat !!! Bilangan Avogadro = 6,02 × 10²³.

X = n . L

= 0,1 . 6,02 × 10²³

= 0,602 × 10²³

= 6,02 × 10²² molekul NO₂ ✔

Jadi, jumlah partikelnya adalah 6,02 × 10²² molekul NO₂.

Semoga membantu.

30. Nitrogen dan oksigen dapat membentuk senyawa NO dan NO2.jika massa N dalam NO dan NO2 adalah sama dan massa O dalam NO sebesar 5 gram,massa O pada NO2 adalah


Nitrogen dan oksigen dapat membentuk senyawa NO dan NO2 sesuai persamaan berikut:

N2 + O2 ⇒ 2NO

N2 + 2O2 ⇒ 2NO2

Maka perbandingan massa N dan O dalam senyawa NO adalah N : O = 1 : 1. Sedangkan dalam NO2 adalah N : O = 1 : 2. Jika massa N dalam NO dan NO2 adalah sama dan massa O dalam NO sebesar 5 gram, maka massa O pada NO2 adalah 10 gram.

O : O = 1 : 2

5 : O = 1 : 2

O = 10 gram

Pembahasan

Dalam suatu reaksi kimia jika jumlah massa dari senyawa-senyawa reaktan tidak sama dengan jumlah produk yang terentuk maka akan menyisakan zat sisa. Sedangkan fraksi senyawa yang lain akan habis duluan. Fraksi senyawa habis secara keseluruhan dalam reaksi disebut sebgaia pereaksi pembatas. Misalnya pada pembentukan Na2SO4 dari 2 mol NaOH dan 2 mol H2SO4 berikut

              2NaOH +  H2SO4 ⇒ Na2SO4 + 2H2O

Mula    :    2 mol       2 mol          -               -

Reaksi :    2 mol       1 mol          1 mol       1 mol

Akhir   :       -            1 mol           1 mol       1 mol

Dalam keadaan setimbang terbentuk 1 mol Na2SO4 dan 1 mol H2O serta zat pereaktan sisa yaitu H2SO4 sebanyak 1 mol. Seangkan NaOH berperan sebagai pereaksi pembatas karena terkonsumsi habis setelah reaksi mencapai keadaan setimbang.

Prinsip pereaksi pembatas ini didasarkan pada hukum perbandingan tetap. Hukum ini menyatakan bahwa perbandingan massa dan volume suatu unsur-unsur pembentuk senyawa sebelum dan sesudah reaksi adalah tetap.

32. Dalam 4,6 gram NO2 (Ar N = 14; O = 16) terdapat ... molekul NO2.Dalam 4,6 gram NO2 (Ar N = 14; O = 16) terdapat ... molekul NO2.


Dalam 4,6 gram NO₂ (Ar N = 14 ; O = 16) terdapat 6,02 × 10²² molekul NO₂

Pembahasan

Mol adalah satuan yang digunakan untuk menyatakan jumlah partikel yang terkandung dalam sejumlah zat.

Kali ini kita akan bahas, Hubungan Mol dg Massa dan Hubungan Mol dg Jumlah Partikel.

Dalam hal ini, terdapat ketetapan, yaitu bilangan Avogadro yang menyatakan 1 mol = 6,02 × 10²³ partikel.

Rumus yg digunakan pada Hubungan Mol dg Massa :

gr = n × Mr atau Ar

gr = n × Mr atau Ar n = gr / (Mr atau Ar)

Keterangan :

Massa = gr

Mol = n

Mr = massa molekul relatif

Ar = massa atom relatif

Rumus yang digunakan dalam Hubungan Mol dg Jumlah Partikel :

X = n . L

Keterangan :

X = jumlah partikel

n = mol

L = bilangan Avogadro (6,02 × 10²³)

~~~~~~~~~~~~~~~~~~

Diketahui :

gr = 4,6 gram NO₂

Ar N = 14

Ar O = 16

Ditanya :

Jumlah partikel (X) ?

Jawab :

n = gr / Mr

= 4,6 / (14 + 2 × 16)

= 4,6 / 46

= 0,1 mol

Maka,

X = n . L

= 0,1 . 6,02 × 10²³

= 0,602 × 10²³

= 6,02 × 10²² molekul ✔

Jadi, jumlah partikelnya adalah 6,02 × 10²² molekul.
