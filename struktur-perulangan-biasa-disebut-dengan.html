5. Sebut dan jelaskan bagian-bagian pada struktur kontrol perulangan!​


Jawaban:

Ada tiga macam jenis dari struktur kontrol pengulangan (loop) yaitu :

1. While

2. Do-while

3. For loops

1.) While

Pernyataan while loop adalah pernyataan atau blok pernyataan yang diulang-ulang sampai mencapai kondisi yang cocok.

2.) Do While

Do-while loop mirip dengan while-loop. Pernyataan di dalam do-while loop akan dieksekusi beberapa kali selama kondisi bernilai benar(true). Perbedaan antara while dan do-while loop adalah dimana pernyataan di dalam do-while loop akan dieksekusi sedikitnya satu kali.

3.)Perulangan For

Perulangan for menyediakan sarana mengulang kode dalam jumlah yang tertentu. Pengulangan ini terstruktur untuk mengulangi kode sampai tercapai batas tertentu.

18. Sebutkan dan jelaskan 4 struktur dasar perulangan ? dan bagaimanapengunaannya ?​


Jawaban:

1. for loop

2. while loop

3. do while loop

4. for each

Penjelasan:

1. for loop

- for adalah salah satu kata kunci yang bisa digunakan untuk perulangan

- blok kode yang ada di dalam for akan selalu di ulangi selama kondisi terpenuhi

- syntax

  for (initial statement; kondisi; post statement){

     // blok kode

  }

2. while loop

- di dalam while loop hanya terdapat kondisi, tidak terdapat initial statement

- syntax

 while (kondisi) {

   // blok kode

 }

3. do while loop

- do while mirip dengan while

- bedanya adalah pengecekan kondisi, while loop dilakukan diawal sebelum perulangan dilakukan, sedangkan do while loop dilakukan setelah perulangan dilakukan

- do while loop pasti dilakukan sekali, walaupun kondisi bernilai false

- syntax

  do {

    // bloke kode

  } while (kondisi)

4. for each

- for each biasanya digunakan untuk mengakses seluruh data array secara otomatis

- syntax

 for (int example : arrayName) {

   // blok kode

}

27. sebutkan dan jelaskan 3 macam struktur perulangan​


1. Struktur Perulangan For

Struktur perulangan / pengulangan jenis for biasanya digunakan untuk melakukan perulangan yang telah diketahui banyaknya. Biasanya jenis perulangan for dianggap sebagai jenis perulangan yang paling mudah dipahami.

2. Struktur Perulangan While

Struktur perulangan while adalah perulangan yang melakukan pemeriksaan kondisi di awal blok perulangan. Kita tahu bahwa perulangan hanya akan dilakukan jika kondisi yang didefinisikan terpenuhi (jika kondisi bernilai benar).

3. Struktur Perulangan Do-While

Berbeda dengan struktur while yang melakukan pemeriksaan kondisi di awal blok perulangan, pada struktur do-while kondisi justru ditempatkan di bagian akhir. Hal ini tentu menyebabkan struktur perulangan do-while minimal akan melakukan satu kali proses eksekusi statemen yang akan diulang walaupan kondisi yang didefinisikan tidak terpenuhi (bernilai salah).

31. sebutkan dan jelaskan 3 macam struktur perulangan​


3 macam struktur perulangan​:

Struktur Perulangan For, umumnya digunakan untuk mengeksekusi sejumlah loop yang telah ditentukan. Diakui secara luas bahwa tipe perulangan for adalah tipe perulangan yang paling sederhana untuk dipahami.Struktur perulangan while adalah perulangan yang memverifikasi kondisi pada awal blok perulangan. Penting untuk dicatat bahwa perulangan hanya akan dijalankan jika kondisi yang ditentukan benar (jika kondisi bernilai benar).Struktur perulangan do-while berbeda dengan struktur while yang memeriksa kondisi di akhir perulangan. Ini berarti bahwa perulangan do-while akan selalu mengeksekusi setidaknya satu pernyataan, bahkan jika kondisi yang ditentukan tidak terpenuhi (nilai salah).

Pembahasan

Perulangan adalah proses mengeksekusi pernyataan dalam suatu program secara terus menerus hingga suatu kondisi terpenuhi untuk menghentikannya. Operasi pengulangan atau perulangan biasanya ada dalam berbagai bahasa pemrograman, karena struktur perulangan sangat membantu dalam meningkatkan efisiensi program.

Dalam C++, ada tiga jenis struktur perulangan, yaitu: struktur For, struktur While, dan struktur Do While. Pada pembahasan kali ini, kita akan membahas ketiga jenis struktur perulangan beserta contoh program dan penjelasannya. Contoh program ini dibuat menggunakan Dev-C++.

1. Struktur Perulangan For

Struktur perulangan / pengulangan jenis for biasanya digunakan untuk melakukan perulangan yang telah diketahui banyaknya. Biasanya jenis perulangan for dianggap sebagai jenis perulangan yang paling mudah dipahami.

Untuk melakukan perulangan dengan menggunakan struktur perulangan for, kita harus memiliki sebuah variabel sebagai indeksnya. Namun perlu sekali untuk diperhatikan bahwa tipe data dari variabel yang akan digunakan sebagai indeks haruslah tipe data yang mempunyai urutan yang teratur, misalnya tipe data int (0,1,2, ... ) atau char ('a' , 'b' , 'c' , ... ).

Adapun bentuk umum dari struktur perulangan for adalah seperti yang tampak dibawah ini:

// Untuk perulangan yang sifatnya menaik (increment)

// Pastikan nialai awal < kondisi saat berjalan

for(variabel = niali_awal ; kondisi_saat_berjalan ; variable++)

{

     Statemen_yang_akan_diulang;

}

// Untuk perulangan yang sifatnya menurun (decrement)

// Pastikan nialai awal > kondisi saat berjalan

for(variabel = nilai_awal ; kondisi_saat_berjalan ; variable--)

{

     Statemen_yang_akan_diulang;

}

Sebagai catatan bahwa jika kita melakukan perulangan yang sifatnya menaik (increment) maka nilai awal dari variabel yang kita definisikan haruslah lebih kecil dari nilai akhir yang dituliskan dalam kondisi (kondisi saat berjalan). Sebaliknya jika kita akan melakukan perulangan yang sifatnya menurun (decrement) maka nilai awal harus lebih besar dari nilai akhir.

Contoh Program dengan Struktur For

Berikut ini contoh program yang menunjukkan perulangan dengan menggunakan struktur for. Program akan ditulis dalam dua tipe, yaitu perulangan for yang sifatnya menaik (increment) dan perulangan for yang sifatnya menurun (decrement).

Contoh perulangan For yang sifatnya menaik (increment)

#include <iostream>

using namespace std;

int main(){

     int MD;

     for(MD=0;MD<8;MD++){

          cout<<"Belajar C++ Bareng MateriDosen.Com"<<endl;

     }

     return 0;

}

Contoh perulangan For yang sifatnya menurun (decrement)

#include <iostream>

using namespace std;

int main(){

     int MD;

     for(MD=8;MD>0;MD--){

          cout<<"Belajar C++ Bareng MateriDosen.Com"<<endl;

     }

     return 0;

}
