35. sebutkan 3 perbedaan perubahan wujud benda yang berbeda


Jawaban:

Perubahan wujud benda terdiri dari penguapan , pengembunan , peleburan , pembekuan , penyubliman dan pengkristalan. Penjelasannya bisa dilihat pada pembahasan dibawah ini.

\texttt{ } \texttt{ }

PEMBAHASAN

Marilah kita ingat kembali beberapa rumus tentang kalor sebagai berikut ini:

\texttt{ } \boxed{Q = m c \Delta t}

Q=mcΔt

\texttt{ } \boxed{Q = m L}

Q=mL

\texttt{ } \boxed{Q = m U}

Q=mU

dimana:

Q = kalor ( Joule )

m = massa zat ( kg )

c = kalor jenis zat ( J/kg°C )

L = kalor lebur zat ( J/kg )

U = kalor uap zat ( J/kg )

Mari kita sekarang mencoba menyelesaikan soalnya.

\texttt{ } \texttt{ }

Perubahan wujud benda dapat dibagi menjadi sebagai berikut ini:

Penguapan → perubahan wujud cair menjadi gas

Pengembunan → perubahan wujud gas menjadi cair

Peleburan → perubahan wujud padat menjadi cair

Pembekuan → perubahan wujud cair menjadi padat

Penyubliman → perubahan wujud padat menjadi gas

Pengkristalan → perubahan wujud gas menjadi padat

\texttt{ }

Penguapan , peleburan , dan penyubliman adalah perubahan wujud benda yang memerlukan kalor atau panas.

Sebaliknya pengembunan , pembekuan , dan pengkristalan adalah perubahan wujud benda yang melepaskan kalor atau panas.
