2. Mozilla Firefox Image
Inilah pesaing berat Internet Explorer ,Firefox.Si Rubah Api ini di buat oleh Mozilla Foundation yang bertujuan untuk software terbuka atau gratis bagi siapapun. Tersedia dalam 50 lebih bahasa termasuk indonesia smile emotikon.Di lengkapi oleh addons atau aplikasi tambah untuk membantu kinerja si rubah api ini. Juga ada versi portablenya.

Web Browser yang pernah terkenal di era tahun 90-an yang paling banyak digunakan sebelum ada IE. Sekitar tahun 1999 Netscape Corporation di akusisi oleh American Online (AOL).Setelah itu Netscape Corporation mendirikan Mozilla Foundation.Jadi bisa diambil kesimpulan bahwa Netscape Navigator adalah browser web yang merupakan gagasan bagi browser Mozilla Firefox.
4. Opera Image
Opera Mini merupakan web browser untuk handphone yang terkenal hingga sekarang. Opera berfitur banyak sekali. Selain gratis browser ini mempunyai versi portablenya.

Safari di produksi oleh Apple Inc (yang meroduksi iPhone dan iPod itu lho).Awalnya Safari dikeluarkan untuk Sistem Operasi Mac saja namun sekitar bulan Juni 2007 di buat untuk Windows.

15. Jelaskan pengertian web server sebagai (hardware) dan (software)​


Jika sebagai Hardware,web server digunakan untuk menyimpan semua data seperti HTML dokumen,gambar,file CSS,Stylesheets,dan file JavaScript.Sedangkan sebagai Software,fungsi web server digunakan untuk pusat kontrol untuk memproses permintaan yang diterima dari browser.

Pembahasan

_________________________

1. Pengertian Web server dan fungsinya

Web Server adalah perangkat lunak yang berfungsi sebagai penerima permintaan yang dikirimkan melalui browser kemudian memberikan tanggapan permintaan dalam bentuk situs web atau lebih umumnya dalam dokumen HTML.

Namun,Web Server dapat mempunya dua pengertian berbeda yaitu sebagai bagian dari perangkat keras Hardware dan peranglat lunak Software (Penjelasannya ada di atas yang Aku jawab).

Fungsi dari Web server adalah :

Memastikan semua modul yang dibutuhkan tersedia dan siap digunakan .Membersihkan penyimpanan,cache,dan module yang tidak terpakai.Melakukan pemeriksaan keamanan terhadap http request yang dikirimkan browser.

2. Asal usul web server

Pada tahun 1990 web server merupakan sebuah proyek yang diusulkannya pada atasannya di CERN (Organisasi Riset Nuklir Eropa)

Bernama CERN httpd yang diusulkan oleh Sir Tim Berners-Lee. Web Server ini berjalan pada server NeXT.

NeXT merupakan perusahaan yang didirikan oleh Steve Jobs setelah keluar dari Apple.

Pada Tahun yang sama ditemukan Browser pertama kalinya yang dinamakan WordWibeWeb.

35. apakah IIS termasuk dalam kategori software web server


IIS atau Internet Information Services atau Internet Information Server adalah sebuah HTTP web server yang digunakan dalam sistem operasi server Windows, mulai dari Windows NT 4.0 Server, Windows 2000 Server atau Windows Server 2003. Layanan ini merupakan layanan terintegrasi dalam Windows 2000 Server, Windows Server 2003 atau sebagai add-on dalam Windows NT 4.0. Layanan ini berfungsi sebagai pendukung protokol TCP/IP yang berjalan dalam lapisan aplikasi (application layer). IIS juga menjadi fondasi dari platform Internet dan Intranet Microsoft, yang mencakup Microsoft Site Server, Microsoft Commercial Internet System dan produk-produk Microsoft BackOffice lainnya.
