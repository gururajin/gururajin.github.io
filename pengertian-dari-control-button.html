26. Apa perbedaan antara control botton window dan control button worksheet


control button window bisa mempengaruhi tampilan worksheet, sedangkan control button worksheet tidak mempengaruhi tampilan window.

Contohnya: bila tombol close pada control window diklik, maka window akan menutup dan worksheet juga akan menutup. Sedangkan bila tombol close pada control button worksheet diklik maka worksheet akan tertutup, namun window masih terbuka.

Pengertian dan tujuan dari Close Button

Close button (tombol keluar) merupakan suatu tombol yang bisa digunakan oleh pengguna untuk keluar dari suatu program aplikasi, saat tak lagi menggunakannya dan ingin mengerjakan kepentingan yang lain.


Apa tujuannya? Untuk keluar dari program aplikasi tersebut, saat sudah lagi tak digunakan.


Ciri-ciri dari close button, seperti :

- Berada di pojok kanan atas

- Icon bertanda "X"

- Dengan box berwarna merah


Di beberapa software aplikasi bisa dengan menekan Ctrl + W untuk menutup atau keluar dari program tersebut.

38. tombol yang terdapat dalam control size button


1. Minimize merupakan tombol untuk memperkecil tampilan jendela. Apabila tombol ini di klik maka tampilan akan hilang, tetapi masih ada dalam Taskbar.
2. Maximize merupakan tombol untuk memperbesar tampilan jendela. Apabila tombol ini diklik maka tampilan akan penuh di layar kemudian akan muncul tombol Restore Down. Tombol ini (restore down) apabila di klik maka tampilan jendela akan kembali diperkecil.
3. Close yang berfungsi untuk menutup jendela

40. Apa pengertian dan fungsi sizing button?


Tombol pengatur jendela tampilan (sizing button) terletak di sebelah atas kanan. Tombol ini berguna untuk mengatur tampilan yaitu tampilan menjadi kecil dan berada pada taskbar, menjadi besar kembali, dan menutup program.
