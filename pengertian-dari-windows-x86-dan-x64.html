1. kenapa windows 10 32bit disebut x86 sedangkan 64 bit disebut x64 mengapa yang 32bit tidak bisa dibilang x32?


Karena seluruh prosesor intel dibuat berdasarkan desain/arsitektur 386, dan karena itulah kode i386 sinonim dengan prosesor Intel 32-bit. Kode x86 adalah sebutan Microsoft pada Windows untuk arsitektur prosesor 32-bit. Saat 64-bit akan dibuat Intel ingin berbeda arsitekturnya, dan lahirlah seri Itanium dan dikenal dengan kode IA-64.

Jadi kesimpulannya nama atau kode i386 / IA-32 / x86 adalah 32-bit, sedangkan EM64T / amd64 /x86-64 / x64 adalah 64-bit, dan IA-64 merupakan untuk Itanium.

Download Media Creation Tool

Download Media Creation Tool

Langkah 2: Jalankan aplikasi tersebut di komputer Windows 10 yang ingin Anda upgrade ke Creators Update. Klik tombol Accept untuk menerima license agreement.

License terms Windows 10 Media Creation Tool

License terms Windows 10 Media Creation Tool

Windows 10 Media Creation Tool

36. cari pengertian windows dan android​


— Windows adalah sebuah sistem operasi yang dikembangkan oleh perusahaan Microsoft. Ini adalah salah satu sistem operasi paling umum digunakan di dunia, terutama pada komputer pribadi (PC) dan laptop. Windows memiliki berbagai versi, seperti Windows 7, Windows 10, dan Windows 11 (hingga tanggal pemotongan pengetahuan saya pada September 2021). Sistem operasi Windows dirancang untuk berbagai keperluan, termasuk penggunaan rumahan, bisnis, dan pengembangan perangkat lunak.

— Android adalah sistem operasi yang dikembangkan oleh Google, dan banyak digunakan pada perangkat seluler seperti smartphone dan tablet. Android juga digunakan pada perangkat lainnya seperti televisi pintar, konsol permainan, dan banyak perangkat berbasis layar sentuh. Android bersifat open source, yang berarti kode sumbernya dapat diakses oleh pengembang, memungkinkan mereka untuk membuat berbagai aplikasi yang berjalan pada perangkat yang menjalankan sistem operasi ini. Android juga memiliki berbagai versi, dengan nama-nama seperti Jelly Bean, KitKat, Lollipop, dan seterusnya.
