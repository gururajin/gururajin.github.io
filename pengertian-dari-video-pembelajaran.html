35. pengertian dari video mkv adalah​


MKV (Matroska Video) adalah format file video yang digunakan untuk menyimpan video, audio, dan subtitle dalam satu file. Format ini dikembangkan sebagai alternatif untuk format file video lain seperti AVI, MP4, dan MOV, dan digunakan secara luas di seluruh dunia untuk menonton film, video musik, dan konten multimedia lainnya. Keuntungan dari format MKV adalah dapat menyimpan banyak jenis file audio dan subtitle dalam satu file video, serta memiliki kualitas video yang sangat baik dengan ukuran file yang relatif kecil.
