17. Didalam Operator PHP Apa Pengertian dari Operator Tipe ?


Pengertian Tipe Data

Tipe data adalah himpunan nilai yang dapat dimiliki oleh sebuah data. Tipe data menentukan apakah sebuah nilai dapat dimiliki sebuah data atau tidak, serta operasi apa yang dapat dilakukan pada data tersebut. Contoh tipe data dalam dunia nyata adalah bilangan bulat.

Jika sebuah data, misalnya umur harus berupa bilangan bulat maka dapat dipastikan bahwa 25, 13, 7 dapat menjadi nilai umur, sedangkan angka yang menggunakan koma seperti 7.5, 19.655 bukan merupakan contoh dari nilai umur.

Contoh bilangan bulat ini dapat kita lihat dalam kasus sehari - hari khususnya dalam hal pencacahan (Ingat kembali bilangan cacah : 1, 2, 3, 4, ... yang merupakan himpunan bagian dari himpunan bilangan bulat).

Contoh :

Contoh 1 : Jumlah siswa dalam kelas ada 20. Angka 20 tersebut adalah bilangan bulat. Tidak akan ditemukan pernyataan : jumlah siswa dalam kelas ada 20,5.

Contoh 2 :Jumlah mobil yang diparkir di tempat parkir. Kita akan menggunakan bilangan bulat dalam kasus ini. Tidak pernah akan kita gunakan angka angka 50,33 atau 40/7 sebagai jumlah dari mobil yang sedang parkir.

Contoh 3 : Selain itu, misalnya data nama seseorang yaitu "Bambang Pamungkas" yang merupakan sebuah deretan huruf dan lain sebagainya.


Dalam sebuah program, setiap variabel dan konstanta memiliki tipe data yang harus dideklarasikan di awal program. Pendeklarasi tipe data tersebut bertujuan untuk menentukan besarnya tempat dalam memori yang akan digunakan untuk menyimpan data pada tersebut saat program dijalankan.

Tipe data dasar adalah tipe data yang dapat langsung digunakan. Secara umum terdapat 2 tipe data dasar, yaitu numerik dan kategorik. Tipe data numerik terdiri atas angka/ kumpulan angka serta dapat mengalami operasi perhitungan, sedangkan tipe data kategorik dapat berupa angka maupun huruf namun tidak dapat mengalami operasi perhitungan.

22. Apa perbedaan dari oop dan prosedural?


OOP (Object-Oriented Programming) dan prosedural adalah dua paradigma atau cara berpikir yang digunakan dalam pemrograman komputer. Berikut adalah beberapa perbedaan utama antara OOP dan prosedural:

OOP menekankan pada abstraksi objek, sedangkan prosedural menekankan pada abstraksi prosedur atau langkah-langkah.

OOP menggunakan konsep pewarisan, dimana suatu kelas dapat mewarisi sifat dan perilaku dari kelas lain. Sedangkan prosedural tidak menggunakan konsep pewarisan.

OOP menggunakan konsep encapsulasi, dimana data dan perilaku disimpan bersama-sama dalam suatu objek. Sedangkan prosedural tidak menggunakan konsep encapsulasi.

OOP menggunakan konsep polimorfisme, dimana suatu objek dapat memiliki beberapa bentuk atau implementasi yang berbeda. Sedangkan prosedural tidak menggunakan konsep polimorfisme.

OOP lebih cocok digunakan untuk mengelola kompleksitas yang tinggi, sedangkan prosedural lebih cocok untuk mengelola kompleksitas yang rendah.

Jadi, dapat disimpulkan bahwa OOP lebih fleksibel dan lebih mudah dipahami daripada prosedural, karena menggunakan konsep-konsep yang lebih abstract dan lebih dekat dengan cara kerja manusia. Namun, demikian, tergantung pada kondisi dan kebutuhan tertentu, salah satu paradigma tersebut mungkin lebih cocok digunakan daripada yang lainnya.

36. Apa itu pseudo code dan apa contoh penerapannya


Jawaban:

Pseudo code adalah sebuah cara untuk menyampaikan algoritma menggunakan bahasa yang mudah dimengerti, tetapi tidak dalam bahasa pemrograman yang spesifik. Pseudo code cocok untuk menyampaikan algoritma kompleks dengan cara yang intuitif dan mudah dimengerti. Contoh penerapannya adalah:

LOOP

 INPUT "Masukkan nilai a"

 INPUT "Masukkan nilai b"

 IF a > b

   OUTPUT "Nilai a lebih besar dari b"

 ELSE

   OUTPUT "Nilai a lebih kecil dari b"

 ENDIF

END LOOP
