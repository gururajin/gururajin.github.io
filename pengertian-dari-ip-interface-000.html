18. Apa fungsi dari IP ,router, switch, interface, access list​


IP address bisa digunakan sebagai alat atau media untuk mengidentifikasi host atau fitur antarmuka pada sebuah jaringan komputer.

Router memiliki fungsi untuk mengirimkan paket data atau suatu informasi melalui internet atau jaringan dari lokasi tertentu ke jaringan lainnya.

Switch mempunyai fungsi umum sebagai suatu concentrator yang dapat menerima serta membagikan data antara sejumlah perangkat komputer.

User Interface berperan sebagai penghubung yang memudahkan penggunanya agar dapat menjalankan suatu program, aplikasi, maupun mengunjungi website.

Access List merupakan sebuah tool yang digunakan di sisi AP (Access Point) untuk melakukan filtering koneksi dari client.

Untuk dapat melakukan koneksi ke internet dan mengubah IP privat client menjadi IP publik, terdapat beberapa langkah yang harus dilakukan.langkah-langkah umumnya:

1. Pastikan bahwa PC client terhubung dengan interface publik yang sudah tersambung ke internet.

6. Jika koneksi ke internet berhasil, maka PC client sudah dapat menggunakan IP publik pada interface publik.

29. Jika terdapat ebuah cript router abc ip router 192. 168. 100. 0 255. 255. 255. 252 192. 168. 100. 5 jika router adalah router bca maka 192. 168. 100. 0 merupakan.


Jawaban:

Dari informasi yang disampaikan, 192.168.100.0 merupakan alamat IP jaringan dari router BCA. Alamat IP tersebut merupakan alamat IP dari jaringan yang dikontrol oleh router BCA, dan biasanya digunakan sebagai alamat IP dari jaringan internal yang terhubung ke router BCA.

255.255.255.252 merupakan subnet mask yang digunakan untuk menentukan jumlah alamat IP yang tersedia di jaringan tersebut. Subnet mask tersebut menunjukkan bahwa terdapat hanya 4 alamat IP yang tersedia di jaringan tersebut, yaitu 192.168.100.1, 192.168.100.2, 192.168.100.3, dan 192.168.100.4.

192.168.100.5 merupakan alamat IP default dari router BCA. Alamat IP tersebut biasanya digunakan untuk mengakses web-based interface atau interface lainnya pada router BCA.

Dari script yang di sampaikan, 192.168.100.0 merupakan alamat IP subnet mask untuk jaringan yang terhubung ke router abc. Subnet mask ini digunakan untuk menentukan bagian dari alamat IP yang digunakan untuk mengidentifikasi subnet atau jaringan yang terhubung ke router. 255.255.255.252 menunjukkan bahwa hanya terdapat 2 bit yang digunakan untuk menentukan subnet, sehingga hanya terdapat 4 subnet yang mungkin terhubung ke router abc. 192.168.100.5 merupakan alamat IP yang digunakan oleh router abc itu sendiri.

Jika router adalah router bca, maka 192.168.100.0 merupakan alamat IP subnet mask yang sama untuk jaringan yang terhubung ke router bca. Namun, alamat IP yang digunakan oleh router bca sendiri mungkin berbeda dari alamat IP yang digunakan oleh router abc.

Pada kasus permasalahan, diminta WLAN dan Port Ether2 di bridge, konsep bridge sendiri yaitu dapat menghuungkan jaringan/port yang berbeda dan menjadikannya dalam satu jalur koneksi, dalam hal ini WLAN dan Ether2 menjadi 1 koneksi. Maka dari itu, router akan membuat interface baru dibawah bridge menu, contohnya pada mikrotik, ketika kita ingin menjadikannya bridge, maka kita akan membuat interface bridge terlebih dahulu, kemudian mengisikan port yang ingin disatukan. Agar lebih mudahnya, berikut ini merupakan langkah-langkahnya (penulsi menggunakan router Mikrotik RouterOS.

1. Masuk pada menu Bridge, lalu klik tanda (+), kemudian isikan nama interface bridge, setelah itu klik ok

2. Masuk ke tab Ports, lalu isikan interface WLAN dan Ether2

3. Kemudian masuk ke menu IP > Address List, kemudian klik tanda (+), tambahkan IP yang sesuai dan pilih interface bridge (nama bridge yang dibuat tadi).

4. Kemudian, masuk ke menu IP > DHCP Server > klik tanda (+) > isikan name sesuai dengan keinginan, kemudian pilih interface bridge (nama bridge yang telah dibuat)

5. Maka dalam hal ini, bridge yang telah dibuat telah berisikan IP dan sebagai DHCP server

Kesimpulan,

Dari pertanyaan yang diberikan, maka WLAN dan Ether2 di bridge, maka interface yang diberikan IP dan setting DHCP Server adalah interface Bridge.

39. bagaimana cara mengatasi printer canon ip 2700 series error 0×00000709 ?​


1. Mengubah User Account Default Dengan Yang Baru

Cara pertama yang dapat kita lakukan untuk menghilangkan pesan error 0x00000709 printer dan bisa kembali membuat set default printer adalah dengan mengubah User Account Komputer Anda. User Account yang selama ini Anda gunakan harus diubah dengan User Account yang baru. Alasannya karena User Account tersebut sudah mengidap penyakit yang menyebabkan error 0x00000709 printer.

Untuk mengubah sekaligus membuat User Account baru caranya mudah, silakan Anda pergi ke Control Panel. Mengaktifkan Control Panel bisa melalui Start kemudian Control Panel atau dengan jalan pintas Run dengan cara menekan Windows+R pada keyboard kemudian ketik Control Panel > Enter.

Jika layar kerja Control Panel sudah aktif, silakan Anda pilih Add or remove user accounts. Fitur ini selain bisa membuat User Account baru, kita juga bisa menghapus User Account.

Perlu diperhatikan sebelum Anda membuat User Account baru, silakan Anda lihat apakah User Account mode Administrator pada komputer Anda terdiri lebih dari satu? Jika terdapat dua atau lebih User Account mode Administrator maka disini Anda tidak perlu membuat User Account baru lagi. Anda cukup menggantikan posisi User Account yang aktif.

Saya asumsikan komputer Anda tidak mempunyai User Account mode Administrator lagi selain User Account yang sedang digunakan sekarang. Maka tugas Anda sekarang adalah membuat lagi User Account. Caranya silakan Anda klik pilih Create a new account.

Silakan Anda ketik nama User Account baru yang akan dibuat pada kolom New account name kemudian Anda pilih mode Administrator. Terakhir jika kedua langkah tadi sudah selesai, klik pilih Create Account.

Sekarang restart (hidupkan ulang) komputer dan ketika komputer Start Up silakan Anda pilih User Account baru yang tadi telah Anda buat. Jika komputer sudah selesai melakukan Start Up-nya silakan Anda coba untuk melakukan set default printer apakah kasus error 0x00000709 masih ada atau tidak? Seharusnya kasus ini sudah selesai karena sekarang komputer Anda sudah menggunakan User Account baru. Jika sukses, silakan Anda hapus User Account awal yang error dari Control Panel seperti yang sudah saya terangkan diawal.
