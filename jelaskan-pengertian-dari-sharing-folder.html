1. jelaskan cara mengatur sharing folder


Sharing folder adalah sebuah fitur pada sistem operasi maupun aplikasi komputasi awan yang memungkinkan seorang pengguna untuk berbagi sebuah folder dengan pengguna lain di waktu yang bersamaan. Dengan demkian, sebuah pekerjaan dapat dikerjakan dengan lebih efektif, tanpa terkendala masalah jarak.

2. Jelaskan cara sharing folder, hardisk dan printer.


Jelaskan cara sharing folder, hardisk dan printer. Sharing folder, hardisk dan printer adalah salah satu fungsi dari adanya jaringan komputer. dengan adanya sharing atau berbagi ini maka kita dapat mempercepat dan menghemat anggaran untuk pengadaan perangkat komputer kita. Misal dengan sharing printer, maka satu printer bisa digunakan oleh komputer lain yang terhubung dengan jaringan kita.

Pembahasan

Setelah mengetahui apa itu sharing folder, hardisk dan printer maka selanjutnya kita akan membahas mengenai bagaimana cara untuk melakukan sharing tersebut. Untuk melakukan sharing folder, sharing hardisk dan sharing printer adalah sebagai berikut

Sharing Folder, cara untuk melakukan sharing folder dapat dilakukan dengan cara mengklik folder yang akan kita share. Kemudian kita pilih Properties kemudian pilih tab menu sharing. Langkah selanjutnya adalah memilih menu Advanced Sharing, beri centang pada menu Share this folder kemudian klik OK.Sharing Hardisk, cara untuk melakukan sharing hardisk sama dengan ketika kita melakukan sharing folder. Hanya perbedaannya adalah kita klik kanan partisi hardisk yang akan kita share. Klik OK Sharing Printer, untuk melakukan sharing printer langkah yang dilakukan sama dengan sharing folder dan sharing hardisk. Untuk mencari printer yang akan dishare, kita bisa memilihnya di menu control panel. Kemudian klik kanan pada printer yang akan dishare. Langkah selanjutnya adalah memilih printer properties dan pilih tab menu sharing. Klik OK.Pelajari lebih lanjut

23. Jelaskan pengertian folder


Folder merupakan suatu program atau tempat yang berguna sebagai alat penyimpanan berbagai jenis file didalamnya. Fungsi utama folder adalah untuk menyimpan file sekaligus mengelompokkannya menurut jenis file itu sendiri. Kita memerlukan beberapa wadah untuk dalam penyimpanan file musik, dokumen dan foto.

Pembahasan

Manajemen  file  adalah  pengelolaan  file  dan  folder  supaya  rapi  dan  mudah dalam penggunaannya. Folder adalah cara yang bagus untuk menjaga komputer tetap teratur. Dapat menggunakannya untuk menyimpan semua file penting di satu tempat.  dapat menggunakannya untuk menyimpanfile-file penting atau dapat menggunakannya untuk menyimpan kertas-kertas yang kurang penting dengan aman. Folder adalah cara yang bagus untuk menyimpan dokumen di satu tempat.

Ada banyak jenis folder yang dapat gunakan. Beberapa yang paling populer termasuk:

1. My Documents: Ini adalah folder yang biasanya terletak di hard drive komputer Anda. Ini adalah tempat yang bagus untuk menyimpan semua file pribadi Anda.

2. Desktop: Desktop adalah folder lain yang terletak di hard drive Anda. Di sinilah Anda dapat menyimpan pintasan ke program yang paling sering Anda gunakan.

3. Unduhan: Folder unduhan adalah tempat semua file yang Anda unduh dari internet disimpan.

Keuntungan berbagi folder adalah:

– Kelola folder/drive dengan mudah di jaringan Anda

– Berbagi folder menghilangkan kebutuhan untuk menyalin file/folder

-Komputer di gudang. kemudian untuk membukanya untuk komputer lain

– Pengirim dan penerima tidak perlu bertemu untuk berbagi file dan folder

- Berbagi file dengan mudah dengan jumlah pengguna lain yang tidak terbatas.

Hilangnya folder dan file yang ada di folder bersama
