6. sebutkan karakterristik static ram​


Penjelasan:

Lawan dari Dynamic RAM adalah Static RAM (SRAM). SRAM (Static RAM), sama halnya dengan DRAM adalah salah satu tipe dari RAM. Perbedaannya dengan DRAM adalah data yang disimpan dalam SRAM bersifat static, sehingga selama teraliri power listrik, maka data akan tetap utuh, berbeda dengan DRAM yang secara periodik harus di refreshq. Selain itu SRAM digunakan untuk Chace Memory (baikitu didalam maupun diluar chip), sedangkan DRAM digunakan untuk Main Memory, sehingga kecepatan SRAM lebih cepat dari DRAM. Static RAM biasanya digunakan pada Cache Memory & Cache Buffer. Static RAM berharga mahal karena bekerja super-cepat dalam mentransfer data.

SRAM didesain menggunakan transistor tanpa kapasitor. Tidak adanya kapasitor membuat tidak ada daya yang bocor sehingga SRAM tidak membutuhkan refresh periodik. SRAM juga didesain menggunakan desain kluster enam transistor untuk menyimpan setiap bit informasi. SRAM tidak boleh dibingungkan dengan memori baca saja dan memori flash, karena ia merupakan memori volatil dan memegang data hanya bila listrik terus diberikan. Chip SRAM lazimnya digunakan sebagai chace memori , hal ini terutama dikarenakan kecepatannya. Saat ini SRAM dapat diperoleh dengan waktu akses dua nano detik atau kurang , kira kira mampu mengimbangi kecepatan processor 500 MHz atau lebih.


7. Cara kerja static dischange pada pesawat terbang


 Yang mana alat ini mampu menangkal efek dari sambaran petir pada pesawat modern saat ini. Alat Yang sepertinya sederhana ini memiliki fungsi yang cukup penting dalam membantu keamanan pesawat, ibarat dia adalah "perisai" petir. yang akan menetralisir 
muatan petir yang mengenai pesawat saat mengudara.

Ketika petir menyambar pesawat, listrik akan mengalir sepanjang alumunium menuju permukaan yang lebih runcing yaitu di ujung-ujung sayap maupun ekor pesawat, nah pada ujung yang lebih lancip itu, muatan listrik akan terlucut berupa percikan api. Jika Static Discharge System Pada Pesawat mengalami Kerusakan atau Malfungsi, maka efek dari sambaran petir akan berakibat tidak baik untuk pesawat, bisa ringan hingga fatal akibatnya.

20. fungsi static library


Static library (juga dikenal sebagai archive) terdiri dari rutinitas yang disusun dan dihubungkan langsung ke program anda. Ketika anda mengkompilasi sebuah program yang menggunakan static library, semua fungsi dari static library menjadi bagian dari eksekusi. Pada Windows, static library biasanya memiliki ekstensi .lib, Sedangkan di linux, perpustakaan statis biasanya memiliki ekstensi .a (archive). Satu keuntungan dari static library adalah bahwa anda hanya harus mendistribusikan executable agar pengguna untuk menjalankan program anda. Karena library menjadi bagian dari program anda, ini memastikan bahwa versi yang tepat dari library selalu digunakan dengan program anda. Juga, karena static library menjadi bagian dari program anda, anda dapat menggunakannya sama seperti fungsi yang sudah anda tulis untuk program anda sendiri. Pada sisi negatifnya, karena salinan library menjadi bagian dari setiap executable, maka ini dapat menyebabkan banyak ruang terbuang. Static library juga tidak dapat diperbarui dengan mudah, untuk memperbarui library, seluruh executable perlu diganti.   Kalau ngak salah

26. Attribut dan method apapun yang di-declare kedalam interface akan bersifat protected static dan private final public abstract dan protected static final public static final dan public abstract private abstract dan static final


Attribut dan method apapun yang di-declare kedalam interface akan bersifat public abstract.

Interface adalah sebuah blueprint yang berisi deklarasi method dan atribut, tetapi tidak berisi implementasinya. Semua method dan atribut di dalam interface harus bersifat public dan abstract.

Public berarti bahwa method dan atribut dapat diakses dari mana saja. Abstract berarti bahwa method tidak memiliki implementasi dan harus diimplementasikan oleh kelas yang mengimplementasikan interface tersebut.

Berikut adalah contoh interface:

```java

public interface Animal {

public abstract void speak();

}

```

Interface ini berisi satu method abstract, yaitu `speak()`. Method ini harus diimplementasikan oleh kelas yang mengimplementasikan interface `Animal`.

Berikut adalah contoh kelas yang mengimplementasikan interface `Animal`:

```java

public class Dog implements Animal {

@Override

public void speak() {

System.out.println("Woof!");

}

}

```

Kelas `Dog` mengimplementasikan method `speak()` dengan cara mencetak "Woof!" ke konsol.

Jadi, jawaban yang benar adalah public abstract.

28. karakteristik dari static routing


Penjelasan:

o Static Routing mempunyai Administrative Distance (AD) 1 sehingga akan lebih dipilih dari pada dynamic routing.

o Better security, static routes tidak diadvertise dalam network.

o Use less bandwidth dari pada dynamic routing protocol, karena tidak melakukan pertukaran route.

o No CPU cycles are used to calculate and communicate routes.

o The path a static route uses to send data is known.

o Konfigurasi dan maintenance yang memakan waktu

o Tidak cocok untuk network skala besar.

o Untuk jaringan kecil yang tidak akan terjadi perubahan topologi secara significant

o Routing ke/dari stub network. Stub network adalah jaringan yang diakses hanya mempunyai 1 exit path (karena hanya mempunyai satu neighbor).

o Untuk unknown network menggunakan default route

32. how can static electricity occur? and what factors are supporting or inhibiting the occurrence of static electricity


Jawaban:

Penjelasan:

Static electricity occurs when there is an imbalance of electric charges on the surface of an object. This happens due to the transfer of electrons between objects, leading to an excess or deficit of electrons on the surface, creating a buildup of static charge.

Factors supporting the occurrence of static electricity:

1. **Friction**: The rubbing or contact between two different materials can lead to the transfer of electrons, causing one material to become positively charged and the other negatively charged.

2. **Humidity**: Dry conditions promote the accumulation of static electricity because moisture in the air helps dissipate charges. In contrast, low humidity allows static charges to build up more easily.

3. **Material Properties**: Certain materials, such as rubber and synthetic fabrics, are more likely to hold a static charge compared to materials like metals or conductors, which allow charges to flow freely.

4. **Surface Area**: The larger the surface area of an object, the more opportunities for electrons to move and create static charge imbalances.

Factors inhibiting the occurrence of static electricity:

1. **Conductivity**: Objects made of conductive materials, such as metals, can quickly dissipate static charges, preventing the buildup of static electricity.

2. **Grounding**: Connecting an object to the Earth through a conductor, such as a grounding wire, allows excess charge to flow away, reducing the potential for static buildup.

3. **Moisture**: High humidity levels prevent static electricity from accumulating since moisture in the air helps neutralize charges.

4. **Ionization**: Ionizing equipment or ionizers can release ions into the air, neutralizing static charges on objects and surfaces.

In everyday situations, we often encounter static electricity when rubbing objects like balloons on our hair, clothes sticking together after being taken out of the dryer, or getting a static shock when touching a doorknob. Understanding the factors that contribute to static electricity can help us manage and minimize its effects, especially in industrial settings where static electricity may pose safety and operational risks.

34. jelaskan mengenai static routing


Static routing (Routing Statis) adalah sebuah router yang memiliki tabel routing statik yang di setting secara manual oleh para administrator jaringan. Routing static pengaturan routing paling sederhana yang dapat dilakukan pada jaringan komputer.  Menggunakan routing statik murni dalam sebuah jaringan berarti mengisi setiap entri dalam  forwarding  table  di  setiap router yang berada di jaringan tersebut.Penggunaan  routing  statik  dalam  sebuah  jaringan  yang  kecil  tentu  bukanlah suatu masalah,  hanya  beberapa  entri  yang  perlu  diisikan  pada  forwarding table di setiap router. Namun Anda tentu dapat membayangkan bagaimana jika harus melengkapi forwarding table di setiap router yang  jumlahnya  tidak sedikit dalam  jaringan yang besar.

36. jelaskan Pedoman dan batasan static NAT:​


NAT memiliki batasan statis dan korelasi untuk memandu penggunaannya. Sistem seperti ini sering disebut  NAT statis. NAT statis bekerja dengan  menerjemahkan semua alamat IP yang tidak terdaftar ke alamat IP yang terdaftar. Statis NAT biasanya digunakan untuk komputer yang dimaksudkan untuk diakses secara eksternal pada. NAT statis ini benar-benar membuang-buang alamat IP terdaftar karena semua komputer dipetakan ke alamat IP terdaftar.

Pembahasan

Jadi jika anda memiliki banyak komputer yang terdaftar, tentu saja alamat IP yang tersedia akan lebih terbatas. Kelemahan lain dari NAT statis adalah bahwa kurang aman daripada NAT dinamis. Ini karena  setiap komputer memiliki alamat IP sendiri. pada  akhirnya meningkatkan risiko penyusup masuk langsung ke  jaringan pribadi Anda. Fitur NAT meliputi :

Minimalkan alamat IP duplikat di jaringan Anda. Simpan alamat IP sah yang disediakan oleh Penyedia Layanan Internet (ISP) Anda. Hindari proses readdressing ketika ada perubahan jaringan. Meningkatkan keamanan jaringan. NAT menawarkan fleksibilitas dan kinerja yang lebih besar.
