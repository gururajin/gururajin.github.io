Penjelasan:

[tex] \text{mean = } \frac{ \text{jumah seluruh data}}{ \text{banyaknya data}} \\ \text{mean = } \frac{ \text{40 + 36 + 42 + 38}}{ \text{4}} \\ \text{mean = } \frac{ \text{156}}{ \text{4}} \\ \boxed{\text{ \pink{mean = 39 } }}[/tex]

39

Penjelasan dengan langkah-langkah:

Cara mencari rata-rata = Jumlah data : banyaknya data

40 + 36 + 42 + 38 = 156 : 4 = 39
