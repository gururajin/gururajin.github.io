2. Jelaskan pengertian File dan pengertian Folder​


Untuk menyimpan data maka biasanya akan disimpan berupa file dan diletakkan di folder. Pengertian dari file dan folder adalah:

File adalah arsip. Arsip terdiri dari konten multimedia, data teks, gambar, aplikasi, penginstal, dll. (File memiliki ekstensi sendiri tergantung pada jenisnya).Folder adalah tempat anda menyimpan file dan tidak memiliki ekstensi.

Pembahasan

Folder adalah program atau tempat yang membantu Anda menyimpan berbagai jenis file. Fungsi utama folder adalah untuk menyimpan file dan mengelompokkannya berdasarkan jenis file itu sendiri. Anda memerlukan wadah untuk menyimpan file musik, dokumen, dan foto anda.

Manajemen file adalah organisasi file dan folder untuk kemudahan penggunaan. Folder adalah cara yang bagus untuk mengatur komputer anda. Anda dapat menggunakannya untuk menyimpan semua file penting Anda di satu tempat. Anda dapat menggunakannya untuk menyimpan file penting, atau anda dapat menggunakannya untuk menyimpan dokumen yang kurang penting dengan aman. Folder adalah cara yang bagus untuk menyimpan dokumen di satu tempat.

Ada banyak jenis folder yang tersedia. Beberapa yang paling populer adalah:

My Documents: Ini biasanya folder di hard drive komputer anda. Ini adalah tempat yang sempurna untuk menyimpan semua file pribadi anda.Desktop: Desktop adalah folder lain di hard drive anda. Di sini anda dapat menyimpan pintasan ke program yang paling sering anda gunakan. Unduhan: Semua file yang diunduh dari Internet disimpan di folder Unduhan.

7. Jelaskan Fungsi dari macam-macam file program Ms.Excel


.xls
tipe file jenis ini merupakan format default Microsoft Excel sebelum versi Excel 2007.

.xlt
format untuk template worksheet Microsoft Excel sebelum versi Excel 2007.

.xla
tipe file excel ini merupakan format untuk Excel Add-in sebelum versi Excel 2007.

.xlsx
tipe file ini merupakan format default worksheet Microsoft Excel 2007, yang telah berbasis XML.

.xlsm
tipe file excel model ini merupakan format worksheet Microsoft Excel 2007, yang telah berbasis XML.

.xlsb
merupakan format worksheet Microsoft Excel 2007, yang telah berbasis XML, tapi dikodekan dalam bentuk biner.

Format ini memiliki keunggulan lebih kepat dibuka dan lebih efisien, mengingat tujuan dibuatnya format ini adalah untuk menangani worksheet yang sangat besar yang terdiri atas puluhan ribu baris atau ratusan kolom.

.xltm
Jenis tipe file ini jarang digunakan, type file ini merupakan format untuk template worksheet Microsoft Excel 2007.
(yang telah berbasis XML tapi mampu menyimpan macro VBA. Merupakan pengganti dari format *.xlt.)

.xlam
merupakan format untuk Excel Add-in untuk menambah kemampuan Excel.

27. menjelaskan fungsi file konfigurasi pada sistem komputer(batch,confi file)


Batch file memiliki 2 jenis fungsi yaitu :
1. Fungsi positif :
Bagi sebagian orang yang cinta kedamaian mereka akan memanfaatkan batch file untuk hal-hal yang baik yang diantaranya : untuk membuat program tuneup windows, program administrator windows dll.
2. Fungsi Negatif :
Bagi sebagian para cracker batch file bisa mereka manfaatkan untuk membuat sebuah aplikasi yang di antaranya : virus, program perusak dll.
Batch file biasanya kita buat menggunakan notepad dan untuk mempelajari pemrogramman batch file tidaklah sesulit kita mempelajari bahasa pemrogramman yang lain nya seperti : c, c++, java, vb dll. Pemrogramman batch file merupakan pemrogramman yang klasik dimana semua orang pun bisa mempelajarinya dengan mudah hanya dengan beberapa hari saja. bahkan ada yang bilang mereka bisa menjadi programmer batch file expert hanya membutuhkan 7 hari saja( sudah bisa buat virus, dan program untuk tuneup windows)
