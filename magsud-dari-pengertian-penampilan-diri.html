19. Pengertian pengatur tampilan halaman kerja


Jawaban: Pengatur tampilan halaman kerja adalah fitur yang memungkinkan pengguna untuk mengubah tampilan halaman kerja sesuai kebutuhan. Pengatur tampilan halaman kerja biasanya terdapat pada aplikasi pengolah kata, pengolah angka, dan pengolah presentasi.

Fungsi pengatur tampilan halaman kerja adalah untuk memudahkan pengguna dalam mengedit dan membaca dokumen, lembar kerja, atau presentasi. Pengatur tampilan halaman kerja dapat digunakan untuk mengubah ukuran halaman kerja, margin, orientasi halaman, dan tampilan tata letak.

Berikut adalah beberapa contoh pengatur tampilan halaman kerja:

Ukuran halaman kerja

Pengatur tampilan halaman kerja dapat digunakan untuk mengubah ukuran halaman kerja, seperti ukuran kertas, ukuran margin, dan ukuran font.

Orientasi halaman

Pengatur tampilan halaman kerja dapat digunakan untuk mengubah orientasi halaman, seperti orientasi potret atau orientasi lanskap.

Tampilan tata letak

Pengatur tampilan halaman kerja dapat digunakan untuk mengubah tampilan tata letak, seperti tampilan grid atau tampilan lembar kerja.

Berikut adalah beberapa cara untuk mengatur tampilan halaman kerja:

Menggunakan menu

Pengatur tampilan halaman kerja biasanya dapat diakses melalui menu. Misalnya, pada aplikasi Microsoft Word, pengguna dapat mengatur tampilan halaman kerja melalui menu Page Layout.

Menggunakan pintasan keyboard

Pengatur tampilan halaman kerja juga dapat diakses melalui pintasan keyboard. Misalnya, pada aplikasi Microsoft Word, pengguna dapat mengatur ukuran halaman kerja menggunakan pintasan keyboard Ctrl+P.

Menggunakan dialog box

Pengatur tampilan halaman kerja juga dapat diakses melalui dialog box. Misalnya, pada aplikasi Microsoft Word, pengguna dapat mengatur orientasi halaman menggunakan dialog box Page Setup.

Dengan menggunakan pengatur tampilan halaman kerja, pengguna dapat membuat dokumen, lembar kerja, atau presentasi yang lebih mudah dibaca dan diubah.

23. Jelaskan pengertian penampilan tari


Kegiatan puncak dalam seni tari adalah penampilan tari. Pengertian dari penampilan tari adalah kegiatan memperlihatkan gerak tari dari para penari kepada para penonton pagelaran karya tari.

Pembahasan

Seni tari adalah salah satu bentuk dari seni audiovisual yang karya seninya terbentuk dari gerak tari yang dihasilkan oleh para penari. Dalam seni tari, kegiatan puncaknya adalah penampilan seni tari. Penampilan seni tari ini adalah kegiatan mempertunjukkan gerak tari yang dilakukan oleh para penari kepada para penonton dengan tujuan menyampaikan pesan atau makna yang ada pada karya tari kepada para penonton. Melalui kegiatan ini, para penonton juga bisa memberikan apresiasi dan kritiknya sehingga bisa menjadi pembelajaran bagi para penari untuk menciptakan karya tari yang lebih baik lagi di masa yang akan datang.

Berikut adalah unsur pendukung dalam penampilan tari:

Tata panggung: adanya dekorasi atau hiasan yang digunakan pada panggung yang menjadi tempat para penari melakukan gerak tarinya untuk menyesuaikan bentuk panggung dengan tema tari.Tata cahaya: adanya pencahayaan yang optimal sehingga para penonton bisa lebih mudah untuk melihat keseluruhan gerak yang dilakukan oleh para penari.Tata suara: adanya penggunaan pengeras suara yang baik sehingga iringan lagu bisa didengar oleh para penari dan juga para penonton.
