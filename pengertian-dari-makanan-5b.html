Kurangkan 2a + 5b dari 2a – 5b

==> (2a - 5b) - (2a + 5b)

==> (2a + 5b) - (2a - 5b)

→ ( 2a + 5b ) - ( 2a - 5b )

→ 2a + 5b - 2a + 5b

→ 2a - 2a + 5b + 5b

→ 0 + 10b

→ 10b

40. Bentuk sederhana dari 3a – 4b - 12a + 9b adalah …. A. -9a + 5bB. 9a - 5bC. -9a - 5bD. 9a + 5b ​


~Aljabar

__________

Bentuk sederhana dari 3a - 4b - 12a + 9b adalah - 9a + 5b

PEMBAHASAN

Diketahui

Bentuk aljabar: 3a - 4b - 12a + 9b

Ditanya

Bentuk sederhana

Solusi

= 3a - 4b - 12a + 9b

= 3a + ( - 4b ) + ( - 12a ) + 9b

Jumlahkan suku-suku sejenis

= 3a + ( - 12a ) + 9b + ( - 4b )

= 3a - 12a + 9b - 4b

= - 9a + 5b

∴ Jadi, Bentuk sederhana dari 3a - 4b - 12a + 9b adalah - 9a + 5b

= (3a - 12a) + (-4b + 9b)

= -9a + 5b ✔
