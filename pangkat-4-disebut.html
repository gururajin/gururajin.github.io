15. Bilangan Berpangkat Memahami konsep bilangan berpangkat Pertanyaan: bagaimana kalian dapat menggunakan bentuk pangkat untuk menyederhanakan penulisan sebuah bilangan? Contoh: 3 x 3 x 3 x 3 x 3 x 3 = 36 dapat disebut dengan perpangkatan 3 (-2) x (-2) x (-2) x (-2) = (-2)4 dapat disebut perpangkatan -2 Secara umum perkalian berulang dari suatu bilangan a dapat disebut dengan perpangkatan a


Kita ingat kembali mengenai eksponen. Eksponen merupakan suatu perkalian berulang. Biasanya eksponen merupakan bilangan yang sama dikali berulang sebanyak [tex] n [/tex] kali.

Misalkan [tex] a [/tex] adalah bilangan sembarang. Eksponen dapat ditulis sebagai

[tex] a^n = \underset{n \text{ kali}}{\underbrace{a \times \dots \times a}} \\ [/tex]

Sebagai contoh, [tex] 2^2 = 2 \times 2 [/tex] yang berarti bilangan [tex] 2 [/tex] dikalikan sebanyak [tex] 2 [/tex] kali. Contoh lainnya [tex] (-1)^4 = -1 \times (-1) \times (-1) \times (-1) = 1 [/tex] yang berarti sebanyak empat kali bilangan [tex] (-1) [/tex] dikalikan.

Seperti yang kita lihat di pertanyaan (untuk contoh) lainnya), [tex] 3 \times 3 \times 3 \times 3 \times 3 \times 3 [/tex] dapat dinotasikan sebagai [tex] 3^6 [/tex] dan tentu saja dengan [tex] -2 \times (-2) \times (-2) \times (-2) = (-2)^4 [/tex].

Penjelasan dengan langkah-langkah:

Mengapa hasil dari pangkat 3 ?

⇒ karena, pada soal yang ditanya adalah bilangan pangkat 3. kubik hanya dipakai saat mencari volume, dan ada satuan panjangnya ( misal : km, hm, dam, m, dm, cm, mm )

====================================================

Berikut adalah contoh bilangan yang berpangkat :

2³ = 2 × 2 × 2

Maksudnya adalah angka dua dikalikan dengan angka yang sama sebanyak tiga kali dan menghasilkan bilangan dua pangkat tiga.

Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 4

3² = ( 3 x 3 ) = 9

4² = ( 4 x 4 ) = 16

5² = ( 5 x 5 ) = 25

Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 8

3³ = ( 3 x 3 x 3 ) = 27

4³ = ( 4 x 4 x 4 ) = 64

5³ = ( 5 x 5 x 5 ) = 125
